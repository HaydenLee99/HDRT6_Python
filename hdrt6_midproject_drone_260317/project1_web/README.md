# 2-1. Menu Structure

## 메뉴 구조
### 1. 메인 페이지
- 메인 배너(로고)
- 네비게이션 바
- 서비스 소개
- 로그인 / 로그아웃
### 2. 비행안전조회
- 검색 배너
- 지도
- 분석결과 패널
### 3. 로그인
- 로그인 폼
- 회원가입 폼
### 4. 마이페이지
- 개인정보 조회/수정
- 조회 기록
- 즐겨찾기
### 5. 안내/공지
- 서비스 소개
- 데이터 출처
- 공지사항
